# ECG 心电线设计验证 DVT 清单

## DVT 的目的

- 确认 ECG 心电线设计是否满足客户规格书、设计输入、图纸、BOM、法规和内部工程要求。
- 验证设备端连接器、患者端 snap / clip / banana、线材、屏蔽、标签、包装等设计组合在样品阶段是否可行。
- 识别 EVT 后遗留的设计风险，例如 pin 位错误、线序混淆、运动噪声偏高、接触电阻不稳定、屏蔽断续、弯折寿命不足。
- 为是否进入 PVT 提供工程依据，避免带着未关闭的设计问题进入小批量验证。
- 建立完整 DVT 记录，便于后续客户承认、设计冻结、量产移交和问题追溯。

## DVT 前置条件

- 客户规格书、产品需求、ECG 导联定义、AHA / IEC 颜色体系要求已经确认并受控。
- 设备端 pin 定义已经由客户或主机工程团队书面确认，包含每个 pin 对应的导联名称、屏蔽、识别电阻或保留脚位。
- 患者端端子形式已经明确，例如 snap、clip、banana，且对应导联标识、颜色、尺寸和接触要求已经确认。
- DVT 用图纸、BOM、工艺流程、检验规范和测试方案已经完成版本冻结。
- EVT 阶段重大问题已经关闭，未关闭问题必须列入 DVT 风险项并定义验证方法。
- DVT 样品已经按受控图纸和受控物料制作，并完成内部样品编号和追溯登记。
- 测试设备已经校准或在有效期内，例如万用表、毫欧计、绝缘电阻测试仪、耐压测试仪、拉力机、弯折机、插拔寿命机、ECG 实机或模拟器。
- 测试环境已经记录，包括温度、湿度、测试人员、测试日期、设备编号和样品编号。

## DVT 输入资料

- 客户规格书：确认导联数量、线长、分支结构、颜色体系、患者端类型、设备端接口、包装和标签要求。
- 设计输入清单：确认所有需求都有对应验证项目，不允许只凭经验判断通过。
- 产品图纸：确认总线长、分支长度、剥线尺寸、端子方向、连接器键位、标签位置、公差和关键尺寸。
- BOM：确认线材、连接器、端子、护套、热缩管、标签、包装袋、屏蔽材料和耗材料号。
- Pin 定义表：确认设备端 pin 与 RA / LA / LL / RL / V 或 IEC 对应导联之间的映射关系。
- 颜色定义表：确认 AHA 体系与 IEC 体系不能混用，样品、标签、图纸和包装标识必须一致。
- 可靠性测试规范：确认插拔、弯折、拉力、酒精擦拭、包装运输等测试条件和判定标准。
- ECG 实机验证方法：确认使用的主机型号、模拟器设置、导联模式、波形输出条件和记录格式。
- EVT 问题清单：确认前期异常是否在 DVT 中复测并关闭。

## DVT 样品要求

- 样品必须来自 DVT 试制批，不得使用手工临时改线、返修线或来源不明样品作为正式验证样品。
- 每条样品必须有唯一编号，例如 DVT-001、DVT-002，并记录生产日期、工单号、图纸版本和 BOM 版本。
- 样品数量应覆盖外观尺寸、电气性能、可靠性、消毒清洁、包装验证和实机验证，破坏性测试样品不得重复用于最终外观判定。
- 若包含不同配置，例如 AHA / IEC、3 导 / 5 导 / 10 导、snap / clip / banana，不同配置必须分别取样验证。
- 可靠性测试前后都要记录样品状态，包括外观、线长、接触电阻、导通、屏蔽连续性和 ECG 波形表现。
- 样品标签、颜色、包装必须与正式设计一致，不允许用临时标签代替量产标签进行追溯验证。

## 外观与尺寸验证

- 检查设备端连接器型号、颜色、键位、防呆方向、外壳成型状态是否与图纸一致。
- 检查患者端 snap / clip / banana 的型号、颜色、金属表面、弹性件和接触面是否有氧化、毛刺、变形。
- 检查线材外被是否有压伤、划伤、鼓包、发白、污染、破皮、热缩不良。
- 检查分叉点、护套、热缩管、应力释放结构是否居中、无偏斜、无明显胶水外溢。
- 检查关键尺寸，包括连接器外形尺寸、端子外露长度、护套长度、热缩长度、标签位置。
- 检查所有尺寸记录必须写入 DVT 尺寸记录表，包含实测值、图纸规格、公差、判定和测量工具编号。

## 线长与分支结构验证

- 在无拉伸、无扭曲的自然平铺状态下测量总线长，避免拉直过度造成虚假通过。
- 分别测量每条导联分支长度，例如 RA、LA、LL、RL、V1 至 V6，记录到对应样品编号。
- 检查分叉位置是否符合人体使用场景，分支过短会影响贴片连接，分支过长会增加缠绕和运动噪声。
- 检查左右导联分支是否容易识别，AHA / IEC 颜色和导联标识必须与分支结构一致。
- 检查分支处是否有应力集中，弯曲后不得出现外被开裂、芯线顶出或屏蔽外露。
- 若线长超差，必须判断是裁线尺寸、成型收缩、装配定位还是测量方法导致，并更新异常记录。

## Pin 位与线序验证

- 按设备端 pin 定义表逐一验证 pin 位，确认每个 pin 对应正确患者端导联。
- 对 RA、LA、LL、RL、V 或 C 导联逐点做导通测试，不允许只抽查部分导联。
- 确认设备端屏蔽 pin、识别电阻 pin、保留 pin 或空 pin 状态符合客户定义。
- 检查相邻 pin、芯线与屏蔽、芯线与外露金属之间是否存在短路。
- 对多配置产品分别验证 AHA 和 IEC 线序，不能用一个体系的样品代替另一个体系判定。
- 记录方式应包含 pin 编号、导联名称、线色、患者端端子、导通结果、短路结果和判定。

## 电气性能验证

- 对每条导联进行导通测试，确认导通稳定且无间歇性开路。
- 晃动设备端尾部、分叉点、患者端尾部时同步观察导通状态，确认无瞬断。
- 检查所有芯线之间无短路，芯线对屏蔽层无非设计短路。
- 如产品含识别电阻、跳线或 EEPROM 等识别结构，测量其阻值或识别结果是否符合客户主机要求。
- 测试前后都应记录样品编号、测试仪器编号、测试条件和结果，可靠性后复测不得省略。
- 电气不良必须保留不良样品，不得先返修后再作为 DVT 通过证据。

## 接触电阻验证

- 使用毫欧计或符合规格的低阻测试设备，测量设备端 pin 到患者端 snap / clip / banana 接触点的总接触电阻。
- 每条导联至少测试 3 次，记录最大值、最小值和平均值，避免单次读数误判。
- 对患者端 snap / clip / banana 进行轻微插合、旋转或晃动后复测，确认接触稳定性。
- 检查 snap 弹片弹力、clip 夹持力、banana 插入力是否影响阻值跳变。
- 插拔寿命、弯折寿命、酒精擦拭后应复测接触电阻，确认老化后仍满足规格。
- 若阻值偏高，应区分端子铆压、焊接、镀层、污染、端子配合尺寸和测试夹具接触问题。

## 绝缘电阻与耐压验证

- 明确测试点位，包括导联芯线之间、芯线对屏蔽、芯线对设备端外壳或外露金属。
- 绝缘电阻测试需记录施加电压、保持时间、实测阻值和环境温湿度。
- 耐压测试需记录测试电压、漏电流上限、保持时间和是否出现击穿、飞弧、报警。
- 测试前确认样品干燥清洁，避免水汽、酒精残留或污染导致误判。
- 测试后复查导通和 ECG 波形，确认耐压未造成隐藏损伤。
- 若发生耐压不良，必须隔离样品并检查剥线长度、端子间距、屏蔽修剪、热缩覆盖和连接器内部爬电距离。

## 屏蔽连续性验证

- 检查线缆屏蔽层是否按设计连续连接到设备端屏蔽 pin、接地端或连接器壳体。
- 测量屏蔽层两端连续性和阻值，记录每条样品结果。
- 检查分叉点、剥线处、屏蔽收口处是否存在屏蔽丝断裂、外露、短接到芯线的风险。
- 弯折设备端尾部和分叉点时同步观察屏蔽导通状态，确认无间歇性断续。
- ECG 实机测试时关注工频干扰、基线漂移和运动噪声，判断屏蔽设计是否足够。
- 若屏蔽连续性异常，应追溯屏蔽压接方式、焊接方式、屏蔽尾长度、铜箔或编织层处理工艺。

## ECG 实机波形验证

- 使用客户指定或内部认可的 ECG 主机、心电模拟器和导联模式进行测试。
- 检查所有导联连接后主机能正确识别，不应出现导联错位、导联脱落误报或无法识别。
- 记录标准模拟信号下的波形形态、幅值、心率显示、基线稳定性和噪声情况。
- 对 AHA / IEC 样品分别确认颜色、导联名称和主机显示是否一致。
- 轻微移动线缆、患者端端子和设备端连接器时，观察波形是否出现尖峰、断续或长时间基线漂移。
- 实机验证记录应保存照片、截图或波形导出文件，并标注样品编号、主机型号和模拟器设置。

## 运动噪声验证

- 在 ECG 实机波形输出状态下进行运动噪声评估，不能只用导通测试代替。
- 模拟使用场景，包括线身摆动、分支轻拉、患者端端子晃动、设备端尾部弯曲。
- 对 snap / clip / banana 分别关注接触面微动导致的波形尖峰和基线扰动。
- 记录运动动作、持续时间、波形变化、恢复时间和是否影响主机判读。
- 对屏蔽结构、线材柔软度、分支长度、端子夹持力与噪声表现建立对应关系。
- 若运动噪声超标，应评估线材结构、屏蔽接地、端子接触、分叉应力释放和患者端固定方式。

## 机械可靠性验证

- 机械可靠性测试应覆盖设备端连接器尾部、主线、分叉点、患者端尾部和端子接触区域。
- 测试项目至少包含插拔寿命、弯折寿命、拉力验证，必要时增加扭转、跌落或运输振动。
- 机械测试前记录外观、电气、接触电阻和屏蔽连续性，测试后用同样项目复测。
- 机械测试不得只看外观是否破损，还要确认 ECG 波形、导通和接触电阻是否退化。
- 对破坏性测试样品应贴状态标签并隔离保存，避免误用于客户承认样品。
- 任何机械测试后出现开路、短路、屏蔽断续、端子松动、标签脱落，都要进入 DVT 异常记录。

## 插拔寿命验证

- 设备端连接器按规定次数进行插拔，记录插拔次数、速度、配对母座型号和测试人员。
- 患者端 snap / clip / banana 按实际使用方式插拔或夹持释放，确认接触稳定性和机械磨损。
- 每个阶段点位，例如 0 次、100 次、500 次、1000 次，应记录外观、插入力或手感、导通和接触电阻。
- 检查设备端锁扣、防呆结构、端子镀层和塑胶壳是否磨损、变形或松动。
- 检查 snap 是否扣合松动，clip 是否夹持力下降，banana 是否插合过松或过紧。
- 插拔后 ECG 实机波形必须复测，确认没有因接触磨损造成波形不稳定。

## 弯折寿命验证

- 弯折位置至少包含设备端尾部、分叉点、患者端尾部，必要时增加标签边缘和热缩端部。
- 按规定弯折角度、弯折半径、负载、频率和次数执行，不得只用手工随意弯折代替正式测试。
- 测试过程中应定期检查导通状态，捕捉间歇性断线。
- 测试后检查外被开裂、护套松动、屏蔽外露、端子松动、热缩位移和线芯断裂。
- 复测所有导联的导通、接触电阻、屏蔽连续性和 ECG 波形。
- 若弯折失效，应切开失效位置分析断线位置、屏蔽结构、应力释放长度和材料硬度。

## 拉力验证

- 拉力点位包括设备端连接器与线缆结合处、分叉点、患者端端子与线缆结合处、标签固定位置。
- 拉力方向应模拟实际受力方向，例如轴向拉力、分支拉力和端子拔出方向。
- 记录施加力值、保持时间、样品位移、是否脱落、是否断线、是否外观损伤。
- 拉力后必须进行导通、接触电阻和屏蔽连续性复测。
- 若端子脱落或芯线断裂，应检查压接高度、焊接强度、胶水或成型包覆、线材抗拉结构。
- 拉力测试结果应与图纸或客户规格中的最小拉力要求逐项对应。

## 消毒与清洁验证

- 使用客户指定或内部认可的清洁剂进行验证，重点覆盖耐酒精擦拭，例如 75% IPA 或客户规定浓度。
- 擦拭区域包括线身外被、标签、印字、连接器外壳、患者端端子外壳和分叉处。
- 记录擦拭次数、擦拭压力、清洁剂浓度、擦拭工具和干燥时间。
- 擦拭后检查外被是否发粘、开裂、变色、硬化，标签是否翘边、脱落、模糊或扫码失败。
- 检查患者端 snap / clip / banana 是否因清洁剂残留导致接触电阻升高或接触不稳定。
- 消毒清洁后需复测导通、接触电阻和 ECG 实机波形，确认清洁过程不影响功能。

## 标签与追溯验证

- 检查标签内容是否包含客户要求的信息，例如产品型号、批次号、序列号、生产日期、导联体系、警示符号。
- 确认标签位置与图纸一致，不影响弯折、不遮挡端子、不导致使用时磨损过快。
- 扫描条码或二维码，确认能追溯到工单、物料批次、生产日期、检验记录和 DVT 样品编号。
- 酒精擦拭、弯折、包装摩擦后复查标签可读性和粘附状态。
- AHA / IEC 不同版本应在标签或包装上有清晰区分，防止混料和误发。
- 追溯记录必须能关联到 DVT 测试结果，不能只记录批次而无法定位单条样品。

## 包装验证

- 检查单条包装方式是否避免线缆过度弯折，最小弯曲半径不得损伤屏蔽和芯线。
- 检查包装内是否有防尘、防潮、防端子刮伤措施，患者端 snap / clip / banana 不应互相挤压损伤。
- 检查包装标签内容与产品标签、客户订单、AHA / IEC 体系、型号和数量一致。
- 进行运输模拟或跌落后，复查外观、导通、接触电阻、屏蔽连续性和 ECG 波形。
- 检查客户开箱后是否容易识别导联颜色和端子类型，避免包装缠绕导致使用前损伤。
- 包装验证记录应包含包装照片、标签照片、装箱方式、样品数量和测试后判定。

## DVT 结论

### 结论模板一：通过，可以进入 PVT

- DVT 样品按受控图纸和 BOM 制作，所有计划测试项目已完成。
- 外观、尺寸、线长、pin 位、线序、电气性能、接触电阻、绝缘耐压、屏蔽连续性、ECG 实机波形、运动噪声、机械可靠性、消毒清洁、标签追溯和包装验证均符合要求。
- DVT 异常已全部关闭，关闭证据完整。
- 结论：本项目 DVT 通过，可以进入 PVT 阶段。

### 结论模板二：有条件通过，需关闭指定问题后进入 PVT

- DVT 主要设计目标已验证通过，未发现影响基本功能和安全的重大设计缺陷。
- 当前仍有指定问题未完全关闭，例如非关键尺寸偏差、包装标识优化、个别标签耐久性补充验证。
- 未关闭问题必须列入行动清单，明确责任人、截止时间、复测样品数量和关闭证据。
- 结论：本项目 DVT 有条件通过，需在关闭指定问题并经项目工程、品质和客户或内部评审确认后进入 PVT。

### 结论模板三：不通过，需要重新设计或重新打样

- DVT 中发现影响功能、安全、可靠性或客户使用的重大问题。
- 典型情形包括设备端 pin 定义错误、AHA / IEC 颜色体系错误、ECG 波形不稳定、运动噪声超标、屏蔽连续性失效、弯折寿命不足、插拔后接触电阻超标。
- 当前问题无法通过简单工艺调整或补充检验解决，需要进行设计变更、重新打样和重新验证。
- 结论：本项目 DVT 不通过，需要完成设计修正或重新打样后重新执行相关 DVT 测试。

## 客户承认资料

- DVT 测试报告：包含测试计划、样品信息、测试条件、测试数据、照片、判定和结论。
- 图纸：提供最新受控版本，包含线长、分支结构、pin 定义、颜色体系、标签位置和包装要求。
- BOM：列出所有关键物料料号、规格、供应商、版本和替代料限制。
- Pin 定义与线序验证记录：清楚展示设备端 pin 到患者端导联的对应关系。
- AHA / IEC 颜色确认记录：包含实物照片、导联名称、颜色表和客户要求对应关系。
- 电气测试记录：包含导通、短路、接触电阻、绝缘电阻、耐压和屏蔽连续性结果。
- ECG 实机验证记录：包含主机型号、模拟器设置、波形截图或导出记录、运动噪声观察结果。
- 可靠性测试记录：包含插拔寿命、弯折寿命、拉力、消毒清洁和包装验证结果。
- 异常与改善记录：列出所有 DVT 异常、原因分析、改善措施、复测结果和关闭证据。
- 样品承认记录：客户承认样品必须与 DVT 通过版本一致，并保留样品编号和追溯信息。
