# ECG 心电线图纸与 BOM 清单

## 图纸与 BOM 的目的

- 明确 ECG 心电线从客户需求到量产交付所需的工程输出资料，避免图纸、BOM、样品和实际生产用料不一致。
- 将设备端接口、患者端接口、导联颜色、Pin 位定义、线长、屏蔽、标签、包装等关键信息固定在受控文件中。
- 为工程打样、采购备料、制程工艺、检验、客户承认和后续变更提供统一依据。
- 支持医疗线束项目追溯，确保图纸版本、BOM 版本、样品版本和客户确认资料一致。
- 让新人项目工程师能够按清单逐项核对，减少漏画、漏标、错料、错 Pin、错颜色等常见风险。

## 图纸输入资料

- 收集客户提供的设备端接口定义，包括连接器型号、Pin 数、Pin 序、锁扣方向、插拔方向和设备端 Pin 功能。
- 收集患者端接口要求，确认使用 snap、clip、banana、DIN、针式或其他接口，并确认必须匹配目标电极片。
- 确认导联系统采用 AHA 还是 IEC，导联名称、颜色和标签规则不能混用。
- 确认 ECG 心电线类型，例如 3 导、5 导、10 导或定制导联组合。
- 确认总长、分线长度、分支点位置、允许公差和客户测量基准。
- 确认线材要求，包括芯数、线规、外径、绝缘材料、外被材料、柔软度、耐弯折要求和生物相容性要求。
- 确认屏蔽要求，包括是否单独屏蔽、总屏蔽、屏蔽层接地位置、是否悬空、是否双端连接。
- 确认标签、UDI、批号、序列号、生产日期、Logo、方向标识和法规标识要求。
- 确认包装要求，包括单条包装、盘绕方式、干燥剂、说明书、外箱标签和运输保护要求。
- 确认适用标准或客户规范，例如电气安全、线材阻燃、RoHS、REACH、清洁度和医疗器械相关要求。

## ECG 线束总成图

- 标明产品名称、客户料号、内部料号、图纸编号、版本、日期、单位和比例。
- 画出完整 ECG 心电线总成外观，包括设备端连接器、主线、分线盒、各导联分支线和患者端接口。
- 总成图中必须能看出主线走向、分支方向、患者端接口数量和导联对应关系。
- 标注总长、分线盒位置、分支长度、关键外径、成品允许公差和测量基准点。
- 标明 AHA 或 IEC 导联系统，不能只写颜色而不写采用标准。
- 标出所有标签、热缩管、护套、尾套、应力释放结构和外观方向。
- 标明成品检验项目，例如导通、绝缘、电阻、外观、拉力、标签、包装和数量检查。
- 总成图版本必须与 BOM 版本、样品承认版本一致。

## 线束展开图

- 展开图应清楚显示每根芯线或分支线从设备端 Pin 到患者端接口的连接路径。
- 标注主线长度、分线盒到每个患者端接口的分线长度，二者必须分开标注。
- 标注每根导联的名称、颜色、线规、编号和患者端接口类型。
- 明确分线盒内部或分支处是否需要焊接、压接、注塑、灌胶或热缩保护。
- 标注屏蔽层、排流线、地线和接地点，不得用“按常规处理”等模糊表述。
- 标出剥皮长度、镀锡长度、压接位置、焊点位置和热缩管覆盖范围。
- 展开图中的导联顺序应与 Pin 位表和 BOM 中线材颜色一致。
- 对左右方向、正反面和连接器视角进行说明，避免生产按镜像方向接线。

## 设备端连接器图

- 标明设备端连接器品牌、型号、Pin 数、Key 位、锁扣方向、插入方向和配套端子型号。
- 必须使用客户设备一致的 Pin 定义，不能仅根据连接器标准资料自行推断。
- 图纸中标出正视图或焊线侧视图，并明确视角定义。
- 标注每个 Pin 对应的导联名称、线色、线规、压接端子或焊接方式。
- 检查连接器外壳、端子、密封圈、尾夹、尾套、螺母、卡扣等配件是否全部进入 BOM。
- 标注端子压接高度、拉力要求、插入力、保持力或客户指定检验要求。
- 若设备端有屏蔽壳、金属外壳或接地端，必须标明屏蔽层连接到哪个 Pin 或金属件。
- 对客户专用连接器，应附客户确认图或规格书编号。

## 患者端连接器图

- 明确患者端接口类型，例如 snap、clip、banana、grabber 或定制电极夹。
- 患者端接口必须与目标电极片匹配，确认尺寸、插入力、保持力和接触稳定性。
- 标注每个患者端接口对应导联名称、颜色、标签内容和接口外观。
- 若 snap 有尺寸差异，应标明 3.9 mm、4.0 mm 或客户指定尺寸。
- 若 banana 接头有直径差异，应标明直径、长度、镀层和绝缘套颜色。
- 若 clip 或电极夹有开口尺寸、弹力或材质要求，必须在图纸中注明。
- 标注患者端金属接触件材料和表面处理，例如不锈钢、铜合金、镀镍或镀金。
- 检查患者端应力释放、包胶、热缩、色环和标签是否与总成图一致。

## 分线盒与分支结构

- 标明分线盒外形尺寸、材料、颜色、硬度、表面纹理和成型方式。
- 标注主线进入分线盒和分支线引出的方向、数量、角度和顺序。
- 明确分线盒内部连接方式，包括焊接、压接、PCB 转接、灌胶或注塑固定。
- 标注分支点到患者端接口的每条分线长度，不能只给平均长度。
- 确认分线盒是否需要 Logo、方向箭头、导联标识或上下面区分。
- 检查分线盒是否影响线材弯折半径、拉力测试和清洁消毒要求。
- 若分线盒内有屏蔽终端、排流线或接地结构，必须在图纸上画出并说明。
- 分支结构需避免导联交叉混乱，图纸应能指导生产排线顺序。

## 导联颜色与标识

- 在图纸首页或备注区明确采用 AHA 或 IEC 导联颜色标准。
- AHA 与 IEC 的导联颜色不能混用，同一项目内图纸、BOM、标签、样品必须统一。
- 检查 RA、LA、LL、RL、V 或 C 系列导联名称是否与客户文件一致。
- 每条分支线应同时标注导联名称和颜色，不能只依赖颜色识别。
- 患者端标签内容应与导联名称一致，例如 RA、LA、LL、RL、V1 或 C1。
- 若客户要求本地语言或特殊缩写，应在图纸备注中写清楚。
- 对色环、热缩管、印字、标签纸等标识方式，应明确位置、尺寸和颜色。
- 检查导联颜色在 BOM 中对应的线材或外被颜色是否一致。

## 线长与尺寸标注

- 总长应从设备端指定基准点量到患者端接口指定基准点，基准点必须在图纸中定义。
- 主线长度、分线盒位置、分线长度必须分别标注，不能只给一个总长。
- 每条分支线应单独列出长度和公差，特别是 V1 至 V6 或 C1 至 C6 胸导联。
- 标注连接器外露长度、尾套长度、热缩管长度、标签位置和包装盘绕直径。
- 对剥皮、镀锡、压接、焊接、套管覆盖尺寸应给出制造尺寸和允许偏差。
- 线长公差应按客户要求或公司标准定义，例如小于 500 mm、500 mm 至 1500 mm、超过 1500 mm 的不同公差。
- 图纸尺寸单位应统一为 mm，并在标题栏或备注中说明。
- 检查尺寸链是否闭合，避免总长与各分段长度相加不一致。

## 材料与结构标注

- 标注主线和分支线的导体材料、线规、芯数、绝缘材料、外被材料、外径和颜色。
- 医疗 ECG 线材通常需要柔软、耐弯折、耐清洁剂，图纸应写明相关材料或性能要求。
- 标明外被材料，例如 TPU、PVC、TPE 或客户指定医用等级材料。
- 标注连接器外壳、端子、患者端金属件、分线盒、热缩管、标签和包装材料。
- 对与人体或电极片接触的部件，应确认材料符合客户或法规要求。
- 标明成型方式，例如焊接后包胶、低压注塑、普通注塑、套管热缩或灌胶。
- 检查所有结构件是否有对应 BOM 物料号，避免图纸有结构、BOM 无物料。
- 对替代材料、环保要求、阻燃等级和颜色差异，应在备注或物料规格中明确。

## 屏蔽结构标注

- 明确 ECG 主线是否使用总屏蔽、单芯屏蔽、铝箔屏蔽、编织屏蔽或复合屏蔽。
- 标注屏蔽层材料，例如铝箔、镀锡铜编织、铜箔丝、排流线。
- 屏蔽层连接方式必须明确，包括连接设备端哪个 Pin、金属壳或接地点。
- 明确屏蔽层在患者端是否悬空、剪除、绝缘处理或连接到指定结构。
- 若采用单端接地，应标明接地端在设备端还是患者端。
- 若采用双端连接，应说明客户确认依据，并评估噪声、漏电流和安全要求。
- 标注排流线剥皮长度、焊接位置、套管保护和绝缘处理。
- 检查屏蔽接法是否与客户设备电路、安规要求和样品测试结果一致。

## 标签与追溯标注

- 标签内容应支持追溯，至少确认客户料号、内部料号、批号、生产日期或序列号要求。
- 若客户要求 UDI、二维码、条码、Logo、CE 或其他法规标识，应在图纸中标出内容、尺寸和位置。
- 标注标签材质、颜色、印刷方式、耐酒精、耐磨、耐清洁剂和附着力要求。
- 明确标签贴附位置，例如主线近设备端、分线盒、包装袋或外箱。
- 导联标签应与导联名称、颜色和患者端接口一一对应。
- 检查标签内容与 BOM、包装图、客户承认书、生产工单保持一致。
- 若使用热缩标签，应标注热缩前后尺寸、收缩倍率和安装方向。
- 追溯码规则应能区分生产批次、版本、供应商批次或客户订单。

## 包装图纸要求

- 标明单条 ECG 心电线的盘绕方式、盘绕直径、扎带位置和是否允许折弯。
- 标注内袋规格、材料、厚度、封口方式、标签位置和数量。
- 若需说明书、标签纸、干燥剂或保护帽，应在包装图和 BOM 中同时体现。
- 标明外箱装箱数量、内盒数量、隔板、缓冲材料和外箱标签内容。
- 检查包装方式是否保护患者端 snap、clip、banana 接头，避免运输变形或刮伤。
- 包装标签应包含客户料号、品名、数量、批号、生产日期、版本或订单号。
- 若客户有洁净包装或医疗包装要求，应标注洁净等级、封口检验和有效期。
- 包装图版本应与产品图纸和 BOM 版本受控一致。

## 关键物料清单

- 设备端连接器：检查型号、Pin 数、锁扣方向、Key 位、配套端子、客户设备匹配性。
- 患者端 snap / clip / banana 接头：检查接口尺寸、颜色、镀层、插拔力、与电极片匹配性。
- ECG 主线：检查芯数、线规、外径、外被材料、柔软度、屏蔽结构、总长。
- ECG 分支线：检查导联颜色、导联名称、线规、分线长度、分支顺序。
- 屏蔽层：检查铝箔、编织、排流线材料和屏蔽层连接方式。
- 分线盒：检查外形尺寸、材料、颜色、出线方向、内部固定方式。
- 外被材料：检查 TPU、PVC、TPE 或客户指定材料的医用要求、环保要求和耐弯折性能。
- 热缩管：检查规格、颜色、收缩倍率、覆盖长度和安装位置。
- 标签：检查导联标签、产品标签、追溯标签内容和耐久性。
- 包装袋：检查尺寸、厚度、封口方式、标签位置和洁净要求。
- 说明书或标签纸：检查语言、版本、法规标识、客户料号和包装数量。

## 图纸与 BOM 一致性检查

- 图纸版本、BOM 版本、样品版本和客户确认版本必须一致。
- 图纸中出现的所有连接器、端子、线材、分线盒、标签、套管、包装材料都必须在 BOM 中列出。
- BOM 中所有物料都应能在图纸、规格书或包装要求中找到对应位置。
- Pin 位表中的导联名称、线色、线规、患者端接口必须与展开图一致。
- AHA 或 IEC 导联颜色必须在图纸、BOM、标签和样品中统一。
- 总长、分线长度、剥皮尺寸、热缩位置和标签位置必须与生产工艺文件一致。
- 屏蔽层连接方式必须在图纸、BOM 结构说明和工艺文件中一致。
- 设备端 Pin 定义必须与客户设备规格书逐项核对，必要时由客户书面确认。
- 患者端接口型号必须与目标电极片验证记录一致。
- 关键物料供应商、品牌、规格版本应与客户承认资料一致。

## 常见错误

- 将 AHA 和 IEC 导联颜色混用，例如导联名称按 AHA、颜色却按 IEC。
- 设备端连接器视角未说明，导致 Pin 位镜像接错。
- 只标总长，不标主线长度、分线盒位置和各分支线长度。
- 患者端接口未确认电极片尺寸，样品能装配但量产插拔力不稳定。
- 屏蔽层只写“接地”或“按样品”，未说明单端、双端、悬空或连接 Pin。
- 图纸中有热缩管、标签或保护帽，但 BOM 漏列。
- BOM 中设备端连接器有外壳但漏列端子、尾夹、密封圈或尾套。
- 图纸版本升级后 BOM 未同步升级，生产按旧物料采购。
- 导联标签内容、颜色和患者端接口对应关系不一致。
- 未使用客户设备 Pin 定义，按连接器默认编号自行接线。
- 包装图未说明盘绕方式，导致线材折弯半径过小或患者端接口受压。
- 关键物料替代未经客户或内部验证批准。

## 输出文件清单

- ECG 心电线总成图：包含外观、总长、分支结构、标签、包装参考和版本信息。
- 线束展开图：包含每根导联路径、线色、线规、分线长度、剥皮和屏蔽处理。
- 设备端连接器图：包含连接器型号、Pin 位视角、Pin 定义、端子和屏蔽接法。
- 患者端连接器图：包含 snap / clip / banana 接口规格、颜色、导联标识和电极片匹配要求。
- 分线盒结构图：包含尺寸、材料、出线方向、内部连接和成型要求。
- Pin 位表：包含设备端 Pin、导联名称、患者端接口、线色、线规、屏蔽要求和备注。
- 线长尺寸表：包含总长、主线长度、分线长度、标签位置和公差。
- 材料规格表：包含主线、分支线、外被、屏蔽、端子、连接器、分线盒和标签材料。
- BOM 清单：包含物料类别、物料名称、规格、用量、供应商、关键物料标识和备注。
- 包装图纸：包含内袋、说明书、标签、盘绕方式、装箱数量和外箱标签。
- 关键物料承认资料：包含连接器、患者端接口、线材、屏蔽材料、分线盒和标签等承认文件。
- 图纸与 BOM 一致性检查记录：记录版本、物料、Pin 位、颜色、线长、屏蔽和包装的核对结果。
