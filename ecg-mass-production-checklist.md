# ECG 心电线量产导入清单

## 量产导入的目的

- 确认 ECG 心电线在 PVT 通过后，设计、物料、工艺、检验、测试、包装和追溯均已具备正式量产条件。
- 确认设备端 pin 定义、患者端 snap / clip / banana 结构、AHA / IEC 颜色体系、屏蔽层处理方式和标签信息已经冻结，量产现场不能再按经验临时调整。
- 通过首批量产数据确认线长、分支长度、分线盒位置、接触电阻、运动噪声、ECG 实机波形和包装状态是否稳定。
- 建立量产初期异常反馈和闭环机制，确保项目工程师能快速识别线序错误、接触不稳、屏蔽不一致、颜色体系混用、标签追溯断点等高风险问题。
- 将 PVT 阶段验证通过的做法转化为可重复执行的量产控制要求，避免首批出货后出现批量质量风险。

## 量产前置条件

- PVT 总结报告已批准，PVT 遗留问题已关闭；若存在让步项，必须有客户、质量、工程和项目共同批准的临时控制措施。
- 客户正式确认量产导入窗口、首批数量、出货日期、包装方式、抽检比例和随货文件要求。
- 设备端连接器 pin 定义已冻结，RA、LA、LL、RL、V / C 等导联与 pin 位对应关系不得再变更。
- AHA 或 IEC 颜色体系已由客户确认，同一项目、同一订单、同一批次内不得混用两种颜色体系。
- 图纸、BOM、线序图、工艺流程、SOP、检验规范、测试规范和包装规范均已受控发行。
- 量产使用的供应商、物料批次、治具、测试设备、生产线和作业人员已经完成量产前确认。
- 首批量产所需的 ECG 实机或 ECG 模拟器、接触电阻测试设备、导通测试设备、绝缘 / 耐压测试设备已校准并完成点检。
- 标签批号规则、客户料号规则、生产日期格式、追溯码生成方式和记录保存路径已明确。

## 量产输入资料

- 客户受控图纸：确认总长、分支长度、分线盒位置、连接器方向、标签位置、包装绕线半径和关键公差。
- 产品规格书：确认导通、短路、绝缘、耐压、接触电阻、运动噪声、屏蔽连续性、插拔力和外观要求。
- 设备端 pin 定义表：确认每个 pin 位对应的导联名称、线芯颜色、屏蔽连接方式和测试治具针位。
- AHA / IEC 颜色定义表：确认导联颜色、标签标识、包装标识和检验记录使用同一体系。
- BOM 与 AVL：确认线材、连接器、snap / clip / banana、分线盒、屏蔽材料、热缩管、标签、包装袋和外箱料号。
- 受控工艺文件：包含流程图、PFMEA、控制计划、SOP、作业参数、返工流程和异常升级规则。
- 受控检验文件：包含 IQC、IPQC、OQC、首件确认、巡检、抽检、外观限度样和不良判定标准。
- 受控测试文件：包含导通、短路、绝缘、耐压、接触电阻、屏蔽连续性、运动噪声、ECG 实机波形抽检方案。
- 项目记录模板：包含首件记录、过程巡检表、测试记录、良率统计表、异常单、返工记录、出货检验报告和追溯表。

## 图纸与 BOM 冻结

- 图纸必须标明设备端连接器型号、pin 1 方向、防呆结构、pin 位定义和导联对应关系。
- 图纸必须标明总长、各分支长度、分线盒位置、分叉角度、标签位置、热缩位置和允许公差。
- 图纸必须明确患者端类型为 snap、clip 或 banana，并定义接触面要求、颜色要求和插拔方向。
- BOM 必须包含线材规格、导体结构、屏蔽结构、外被材质、端子镀层、连接器、标签、胶壳、热缩管和包装材料。
- BOM 中的 AHA / IEC 颜色体系必须与图纸、标签和包装一致，不能出现一个文件写 AHA、另一个文件写 IEC 的情况。
- 屏蔽层、排流线、铜箔、编织层、热缩管和绝缘隔离材料的料号必须固定，替代料需走 ECN 或客户批准。
- 客户料号、内部料号、BOM 版本和图纸版本必须一一对应，避免首批出货时料号与实物不一致。

## 工艺文件冻结

- SOP 必须覆盖裁线、剥线、屏蔽处理、焊接 / 压接、分支定位、分线盒组装、热缩、测试、标签、包装等完整流程。
- 裁线工序必须明确总长、分支预留长度、剥线长度和线材批号记录方式。
- 屏蔽处理工序必须明确屏蔽层保留长度、翻折方式、排流线连接点、绝缘隔离方式和热缩覆盖范围。
- 设备端组装工序必须明确 pin 位插入顺序、焊接 / 压接参数、拉力要求和防错确认方法。
- 患者端 snap / clip / banana 组装工序必须明确接触面保护、压接高度、焊接状态、插拔力和污染防护。
- 分支成型工序必须使用定位治具控制线长、分支长度和分线盒位置，不能依靠人工目测。
- 包装工序必须明确绕线直径、扎带位置、端子保护方式、标签朝向和外箱堆码要求。
- 返工流程必须明确哪些不良允许返工、返工后需要复测哪些项目、返工记录如何关联原批号。

## 检验标准冻结

- IQC 必须确认关键物料型号、批号、颜色、端子镀层、线材外被、屏蔽结构和包装状态。
- IPQC 必须确认线长、分支长度、分线盒位置、剥线尺寸、屏蔽处理一致性、热缩位置和标签位置。
- OQC 必须确认外观、尺寸、电性能、ECG 实机抽检记录、标签追溯、包装状态和随货文件。
- AHA / IEC 颜色体系必须作为检验重点，发现混用、错标、错色应按重大不良处理。
- 患者端 snap / clip / banana 需要检查接触面无划伤、无氧化、无污染、无松动、插拔手感稳定。
- 屏蔽层检查需要确认无散丝、无短路、无断开、无排流线虚焊、无热缩覆盖不足。
- 包装检验需要确认线材无过度弯折、分支无折痕、端子无受压变形、标签无脱落或遮挡。
- 检验标准必须定义不良等级、判定样板、抽样规则、隔离要求和升级路径。

## 测试方案冻结

- 导通测试必须 100% 覆盖设备端所有 pin 位与患者端导联的对应关系，不能只确认通断而忽略导联名称。
- 短路测试必须确认任意导联之间无短路，屏蔽层与信号芯线之间无异常导通。
- 绝缘和耐压测试必须按客户规格执行，并记录设备编号、测试参数、测试日期和测试人员。
- 接触电阻测试必须覆盖患者端 snap / clip / banana，重点关注插拔后、轻微晃动后和不同接触角度下的稳定性。
- ECG 实机或 ECG 模拟器抽检必须确认波形稳定、导联识别正确、无明显基线漂移、无导联反接和无异常噪声。
- 运动噪声测试必须模拟线材轻微摆动、分支弯折、患者端端子晃动等实际使用场景。
- 屏蔽与抗干扰测试必须确认屏蔽连续性、屏蔽短路风险和工频干扰表现。
- 测试不合格品必须隔离、贴不良标识、记录样品编号，并按异常流程分析原因。

## 物料与供应链准备

- 关键物料安全库存已确认，首批量产数量、损耗率、返工补料和备品数量已覆盖生产需求。
- 线材批号、连接器批号、患者端端子批号、屏蔽材料批号、标签批号和包装材料批号均需可追溯。
- 供应商来料检验规范已下发，关键尺寸、电性能、颜色体系和包装防护要求已同步供应商。
- AHA 与 IEC 相关物料必须分区存放、分订单领用，避免颜色相近物料混放。
- 设备端连接器需确认 pin 数、键位、防呆方向、端子镀层和插拔配合状态。
- snap / clip / banana 端子需确认型号、接触弹性、夹持力、镀层状态和包装防刮伤措施。
- 屏蔽材料和排流线需确认规格、柔软度、焊接性、连续性和与外被 / 热缩的匹配性。
- 物料替代、供应商切换、批号异常和来料不良必须在上线前关闭，不能带风险进入首批量产。

## 生产线准备

- 生产线布局需按正式流程设置，避免半成品交叉、线材拖地、端子碰伤和 AHA / IEC 物料混放。
- 现场必须摆放受控版本的图纸、线序图、颜色对照表、SOP、检验规范和不良样板。
- 裁线、剥线、屏蔽处理、分支定位、设备端组装、患者端组装、测试、标签、包装工位均已完成点检。
- 分支定位治具需能稳定控制总长、分支长度和分线盒位置，首批量产不得使用未验证的临时治具。
- 屏蔽处理工位需明确工具、操作方向、剪切长度、隔离方式和检查点，确保不同班次做法一致。
- 现场需设置不良品隔离区、待判区、返工区和合格品区，并明确标识。
- 生产节拍、人员配置、检验节拍和测试能力需匹配首批计划，避免因赶工漏检。

## 人员培训确认

- 作业员已培训 ECG 导联名称、设备端 pin 定义、AHA / IEC 颜色差异和患者端端子类型。
- 作业员已培训裁线、剥线、屏蔽处理、焊接 / 压接、热缩、分支成型、标签和包装关键参数。
- 测试员已培训导通、短路、绝缘、耐压、接触电阻、ECG 实机波形和运动噪声测试方法。
- IPQC 已培训线长、分支长度、分线盒位置、屏蔽处理一致性、标签追溯和包装变形检查方法。
- OQC 已培训客户出货标准、抽样规则、随货文件、标签扫描和包装开箱检查要求。
- 所有相关人员已完成培训签到和岗位考核，未培训人员不得参与首批量产关键工序。
- 首批量产期间新换班、新上岗或临时支援人员必须由班组长和工程师进行现场确认。

- 裁线、剥线、分支定位、端子压接、焊接定位、热缩、标签贴附和包装治具已确认状态良好。
- 设备端 pin 位测试治具必须与冻结 pin 定义一致，并能识别 pin 位错插、导联反接和短路。
- 患者端 snap / clip / banana 接触电阻测试夹具必须与端子匹配，夹持稳定且不损伤接触面。
- 导通测试仪、绝缘电阻仪、耐压仪、接触电阻仪、拉力机、卡尺和量具均在校准有效期内。

- 屏蔽连续性和抗干扰抽检所需设备、夹具、测试线和记录模板已确认。
- 测试设备每日点检项目、点检频率、点检人员和异常处理方式已写入现场记录。

## 首件确认

- 首件必须由正式生产线、正式人员、正式物料、正式治具和正式工艺完成，不能由工程师绕过量产流程手工制作。
- 首件需确认设备端连接器方向、pin 1 位置、pin 位定义、导联对应关系和防呆结构。
- 首件需确认 AHA 或 IEC 颜色体系与图纸、标签、包装、测试记录完全一致。
- 首件需测量总长、各分支长度、分线盒位置、标签位置、热缩位置和关键外形尺寸。
- 首件需确认 snap / clip / banana 接触面无损伤、插拔 / 夹持稳定、接触电阻符合要求。
- 首件需确认屏蔽层处理方式一致，包括保留长度、排流线连接、绝缘隔离和热缩覆盖。
- 首件需完成导通、短路、绝缘、耐压、接触电阻、ECG 实机波形和运动噪声确认。
- 首件记录需由生产、工程、质量、项目签字放行；首件未通过不得进入批量连续生产。

## ECG 实机功能抽检

- 每批必须按抽样方案接入客户认可的 ECG 实机或 ECG 模拟器，确认各导联波形显示正常。
- 抽检样品必须覆盖不同生产时间段、不同作业员、不同物料批号或不同生产线。
- 检查设备端插入是否顺畅、锁扣是否可靠、pin 位识别是否正确、导联显示是否对应。
- 检查患者端 snap / clip / banana 与电极片、夹具或模拟端连接是否稳定，无松脱和接触不良。
- 轻微移动线材、摆动患者端端子、弯折分支处，观察是否出现尖峰、断续、基线漂移或接触噪声。
- 保存波形截图、样品编号、测试设备型号、测试日期、测试人员和判定结果。
- 若发现波形异常，需隔离同批产品并追溯 pin 定义、线序、接触电阻、屏蔽处理和作业人员。

## 屏蔽与抗干扰抽检

- 确认屏蔽层处理方式与冻结 SOP 一致，包括单端接地 / 双端处理、排流线焊点和绝缘隔离方式。
- 抽检屏蔽连续性，确认屏蔽回路未因裁切、焊接、热缩、注塑或组装过程断开。
- 抽检屏蔽与信号芯线之间的绝缘状态，防止屏蔽丝散开造成间歇短路。
- 在 ECG 实机或模拟测试环境中观察工频干扰、运动噪声和基线漂移表现。
- 对屏蔽处理不一致的样品按作业员、班次、物料批号、治具和工位进行分层统计。
- 屏蔽异常必须确认是否影响整批产品；必要时暂停出货并增加同批复测比例。

## 标签与追溯管理

- 标签内容必须包含客户料号、内部料号、批号、生产日期、数量、导联体系、追溯码等客户要求信息。
- 产品标签、内袋标签、外箱标签和出货文件中的客户料号、批号、生产日期必须一致。
- 每条或每包产品的追溯码应能关联到线材批号、连接器批号、患者端端子批号、作业员、测试记录和检验记录。
- 标签位置、方向、粘贴牢固度、可读性、扫码结果和耐擦拭状态需按检验标准确认。
- AHA / IEC 标识必须与实际线材颜色体系一致，不能因标签模板错误造成错标。
- 返工品必须保留原始批号和返工记录，不得重新贴标造成追溯链断裂。
- 留样、客户送样、测试样、不良样和报废样均需记录最终去向。

## 包装与出货确认

- 包装方式必须符合客户规范，包括绕线直径、扎带位置、内袋类型、端子保护、说明书、内盒、外箱和运输标签。
- 绕线半径必须足够大，不能造成线材永久弯折、分支折痕、外被压伤或分线盒受力。
- snap / clip / banana 接触面需要防刮、防压、防污染保护，避免运输中互相摩擦。
- 设备端连接器需要防尘、防刮、防 pin 位受压措施。
- 同一包装内不得混装 AHA 与 IEC 产品，不得混装不同客户料号、不同版本或不同批号产品。
- 外箱数量、标签批号、客户料号、生产日期、箱号和装箱清单需与实物一致。
- 出货前需完成 OQC 报告、测试记录、客户要求的 CoC / 检验报告 / 随货资料确认。
- 必要时进行模拟运输、跌落或堆码评估，开箱后确认线材无弯折变形、标签无脱落、端子无损伤。

## 客户反馈跟进

- 首批出货后项目工程师需跟进客户 IQC、上线使用、ECG 实机适配、包装开箱和标签扫描反馈。
- 客户反馈需按批号、客户料号、生产日期、追溯码和出货日期快速定位到生产记录。
- 若客户反馈波形异常，需优先核查 pin 定义、导联对应、接触电阻、运动噪声和屏蔽处理记录。
- 若客户反馈颜色、标签或包装问题，需核查 AHA / IEC 体系、标签模板、包装记录和出货文件。
- 客户退回样品需保持原包装、原标签和问题状态，项目工程师组织工程、质量、生产共同分析。
- 客户反馈关闭后，需更新问题清单、经验教训、控制计划或 SOP，并确认是否影响在制品和库存品。
