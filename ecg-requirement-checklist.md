# ECG 心电线客户需求确认清单

本文面向医疗线束项目工程师新人，用项目工程师的角度，将 ECG 心电线项目启动前需要向客户确认的需求拆解为具体问题，便于需求澄清、报价评估、打样设计和后续量产导入。

## 产品基本信息

1. 该 ECG 心电线用于哪一类设备，例如监护仪、心电图机、动态心电记录仪或遥测设备？
2. 客户是否已有对应设备型号、原厂线缆料号或参考样品可提供？
3. 本项目是新产品开发、原有产品替代，还是现有供应商转移？
4. 预计年用量、单次订单量和目标导入时间分别是多少？
5. 客户是否要求产品满足特定法规、认证或标准，例如 IEC 60601、AAMI、CE、FDA 或 RoHS？
6. 客户是否有目标成本区间或必须匹配的竞品价格？

## 使用场景

1. 该心电线主要用于成人、儿童、新生儿，还是需要覆盖多类患者？
2. 使用环境是病房、手术室、ICU、急诊、救护车，还是家庭护理场景？
3. 产品是一次性使用、单病人重复使用，还是多病人重复使用？
4. 使用过程中是否会频繁移动患者或设备，例如转运监护或移动查房？
5. 客户是否对线缆柔软度、重量、拖拽手感或患者佩戴舒适性有明确要求？
6. 产品是否需要适用于高湿、高温、低温或强电磁干扰环境？

## 设备端接口

1. 设备端连接器类型是什么，例如 DB15、DIN、圆形推拉自锁、插针式或客户自定义接口？
2. 客户能否提供设备端接口图纸、针位定义、外形尺寸和配合公差？
3. 设备端连接器是否需要防呆结构、锁扣、螺丝固定或推拉自锁功能？
4. 设备端插头外壳颜色、材料、Logo、方向标识是否有指定要求？
5. 设备端针脚是否有镀金厚度、接触电阻或插拔寿命要求？
6. 设备端连接器是否需要兼容客户现有设备或第三方品牌设备？

## 患者端接口

1. 患者端采用哪种电极连接方式，例如扣式 snap、夹式 clip、香蕉插、针式或鳄鱼夹？
2. 电极接口规格是多少，例如 snap 直径、香蕉插尺寸或夹子适配电极厚度？
3. 患者端连接器是否需要防触电结构或符合防误插要求？
4. 患者端分支处是否需要颜色套管、号码标识或导联字母标识？
5. 患者端接口是否需要可拆卸、可更换或一次性设计？
6. 客户是否对患者端金属件材料、镀层和皮肤接触安全性有要求？

## 导联类型与颜色体系

1. 客户需要 3 导、5 导、6 导、10 导、12 导，还是其他特殊导联配置？
2. 导联标识采用 AHA 体系、IEC 体系，还是客户自定义颜色和文字体系？
3. 每根分支线需要标注的字符是什么，例如 RA、LA、LL、RL、V 或 R、L、F、N、C？
4. 胸导联是否需要区分 V1-V6 或 C1-C6，并分别使用不同颜色？
5. 客户是否要求颜色与现有设备菜单、说明书或医院使用习惯保持一致？
6. 导联颜色是通过线材本色、彩色套管、标签，还是连接器外壳来实现？

## 线长结构

1. 总线长度从设备端插头到分叉点、从分叉点到患者端分别是多少？
2. 各导联分支线是否等长，还是四肢导联和胸导联需要不同长度？
3. 客户是否有线长公差要求，例如 +/-20 mm、+/-30 mm 或按图纸执行？
4. 分叉结构采用 Y 型、多分支扇出、集线盒，还是模具一体分线结构？
5. 分叉点位置是否需要满足患者佩戴、床边走线或设备固定方式？
6. 是否需要加装线夹、束线带、滑动扣或防缠绕结构？

## 线材与外被材料

1. 主线和分支线的外被材料指定为 TPU、PVC、硅胶、TPE，还是由供应商推荐？
2. 客户对线材柔软度、外径、颜色、表面光泽和触感是否有具体要求？
3. 线材是否需要满足无乳胶、无邻苯、RoHS、REACH 或生物相容性要求？
4. 是否要求低噪声线材，避免患者移动时产生明显运动伪差？
5. 外被是否需要耐酒精、耐清洁剂、耐汗液或耐皮脂腐蚀？
6. 客户是否指定线材供应商、材料牌号或已有材料承认书？

## 屏蔽与抗干扰要求

1. 客户是否要求整线屏蔽、单芯屏蔽、双层屏蔽或无屏蔽结构？
2. 屏蔽层接地方式如何定义，是单端接地、双端接地，还是按设备端针位连接？
3. 是否需要内置导电布、铝箔、编织网或排流线？
4. 客户是否有抗工频干扰、抗射频干扰或抗电刀干扰的测试要求？
5. 设备使用环境是否靠近高频手术设备、无线发射设备或大型电气设备？
6. 客户是否能提供现有产品的噪声指标、波形对比或失败案例？

## 电气测试要求

1. 每根导线的导通关系和针位定义是否已有客户确认图纸？
2. 导通电阻上限是多少，是否按单根导线、整条回路或连接器接触电阻分别判定？
3. 绝缘电阻测试电压、测试时间和最低阻值要求分别是多少？
4. 耐压测试电压、漏电流上限和测试时间是否有明确要求？
5. 是否需要测试屏蔽连续性、屏蔽与芯线绝缘或屏蔽接地电阻？
6. 客户是否要求 100% 出厂电测，并随货提供测试报告或批次记录？

## 机械可靠性要求

1. 设备端和患者端连接器的插拔寿命要求是多少次？
2. 主线、分支线和连接器尾部是否有拉力测试要求，拉力值和保持时间是多少？
3. 线缆弯折测试的弯折角度、负载重量、次数和判定标准是什么？
4. 分叉点、注塑尾套和连接器应力释放结构是否需要特别加强？
5. 产品是否需要通过跌落、扭转、摇摆或线夹夹持测试？
6. 机械测试后是否需要重新进行导通、绝缘、耐压和外观检查？

## 消毒与清洁要求

1. 客户允许使用哪些清洁或消毒方式，例如酒精擦拭、季铵盐、含氯消毒剂或紫外线？
2. 是否禁止使用某些清洁剂，以避免材料开裂、发粘、褪色或印字脱落？
3. 客户要求的清洁循环次数是多少，例如 100 次、300 次或 500 次擦拭？
4. 消毒后是否需要验证外观、导通、绝缘、连接器插拔力和标签可读性？
5. 产品是否需要支持高水平消毒或仅支持日常表面清洁？
6. 客户是否有医院实际清洁流程或清洁剂品牌清单可提供？

## 标签与追溯要求

1. 产品本体是否需要标签，标签位置在主线、分叉盒、设备端还是包装袋上？
2. 标签内容需要包含哪些信息，例如型号、批号、生产日期、UDI、Logo 或方向标识？
3. 标签材料和印刷方式是否需要耐酒精擦拭、耐磨或耐高低温？
4. 客户是否要求每条产品具备唯一序列号，还是只需要批次追溯？
5. 追溯信息需要关联哪些生产记录，例如原材料批号、电测记录、检验员和生产日期？
6. 客户是否提供标签模板、条码规则、二维码内容或 UDI 数据格式？

## 包装要求

1. 单条产品采用单独 PE 袋、纸塑袋、吸塑盒，还是多条合包？
2. 包装是否需要灭菌，若需要，灭菌方式和有效期要求是什么？
3. 包装标签需要包含哪些内容，例如型号、数量、批号、有效期、条码或客户料号？
4. 每箱装箱数量、内盒数量、外箱尺寸和毛重是否有客户要求？
5. 包装内是否需要放置说明书、合格证、干燥剂或保护套？
6. 运输包装是否需要通过跌落、振动、堆码或温湿度测试？

## 客户待确认问题汇总

1. 客户是否能提供完整的产品图纸、针位定义、导联颜色表和参考样品？
2. 客户是否确认目标导联类型、接口规格、线长结构和材料要求作为报价输入？
3. 客户是否明确产品适用标准、认证边界和需要供应商提供的技术文件？
4. 客户是否确认电气测试、机械测试、清洁测试和出厂检验标准？
5. 客户是否确认标签、追溯、包装、装箱和随货文件要求？
6. 客户是否确认项目时间计划，包括图纸确认、样品交付、验证反馈和量产导入节点？
