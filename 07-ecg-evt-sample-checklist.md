# ECG 心电线工程样品 EVT 清单

## EVT 的目的

- 确认 ECG 心电线的设计方案能否做成工程样品，包括设备端连接器、患者端 snap / clip / banana、主线、分支线、分线盒、标签和包装方案。
- 初步验证电气功能是否满足设计输入，重点确认设备端 pin 定义、导通关系、屏蔽层连接方式和患者端接触稳定性。
- 通过实机连接观察 ECG 波形是否稳定，是否存在明显断线、基线漂移、运动噪声或接触噪声。
- 识别设计、BOM、工艺、治具、测试方法中的早期问题，为 DVT 前的设计修正提供依据。
- 建立样品版本、样品编号、测试记录、异常记录和客户送样资料的可追溯链路。

## EVT 前置条件

- 已冻结 EVT 用图纸版本，图纸中必须包含总线长、各分支长度、分线盒位置、剥线长度、端子压接要求、患者端连接形式和标签位置。
- 已确认 ECG 导联体系，AHA 与 IEC 颜色体系必须二选一，不允许同一批样品混用颜色、标签或导联标识。
- 已确认设备端连接器型号、键位、防呆方向、pin 定义和针脚编号读取方向。
- 已确认患者端类型，snap、clip、banana 的规格、镀层、插拔方向和适配电极片或转接件已明确。
- 已完成 EVT BOM 初版，关键物料已有供应商、规格书、样品批次和替代料控制要求。
- 已确认 EVT 测试设备可用，包括导通测试仪、绝缘电阻测试仪、耐压测试仪、卡尺、拉力计、实机 ECG 设备或等效验证平台。
- 项目工程师、研发、工艺、品质和生产已确认样品制作窗口、责任人和记录模板。

## EVT 输入资料

- 客户需求或项目需求清单，包含导联数量、接口类型、目标设备型号、线长要求、颜色体系、消毒要求和预期使用场景。
- EVT 图纸，包含设备端 pin 定义表、线束展开图、分支尺寸、标签内容、关键尺寸公差和特殊工艺说明。
- EVT BOM，包含线材、屏蔽层结构、连接器、端子、患者端 snap / clip / banana、分线盒、热缩管、标签、包装材料。
- 设备端接口资料，包含 mating connector、pin 序号、信号名称、屏蔽 / GND 连接要求和客户设备端识别方式。
- 患者端连接资料，包含电极片或附件规格、接触电阻要求、插拔寿命要求和防脱落要求。
- 工艺流程草案，包含裁线、剥线、屏蔽处理、压接、焊接、注塑或灌封、组装、测试和包装步骤。
- EVT 检验计划，包含外观、尺寸、电气、实机连接、抗干扰、机械可靠性和清洁消毒初步确认项目。

## 样品制作计划

- 明确 EVT 样品批次编号，例如 EVT-01，并在所有图纸、BOM、测试记录和样品标签中使用同一批次编号。
- 制定样品制作排程，至少包含物料到料、首件制作、过程检查、全检、实机测试、异常复盘和客户送样日期。
- 首件样品必须由项目工程师、工艺工程师和品质工程师共同确认后再继续制作后续样品。
- 对高风险工序设置暂停点，例如设备端 pin 装配、屏蔽层处理、患者端压接或焊接、分线盒定位。
- 每根样品制作完成后立即贴样品编号，编号不得只写在包装袋上。
- 计划中需预留拆解样品，避免所有样品都送测后无法分析压接、焊点、屏蔽层和内部结构。

## 物料准备

- 线材需确认导体规格、芯线颜色、屏蔽结构、外被材质、外径、柔软度和供应商批次。
- 设备端连接器需确认型号、pin 数、键位、防呆方向、端子规格、壳体颜色和锁扣结构。
- 患者端 snap / clip / banana 需确认接触面镀层、尺寸、插拔手感、与电极片或附件的匹配性。
- AHA / IEC 颜色套管、标签或印字内容需按项目定义备料，禁止因缺料临时混用另一套颜色体系。
- 分线盒、护套、热缩管、SR、扎带、标识环等辅料需按图纸尺寸和材料要求准备。
- 标签材料需确认耐酒精、耐清洁剂、耐摩擦和打印清晰度，样品编号必须能追溯到版本和制作批次。
- 替代料必须在 EVT 记录中单独标注，并由项目工程师确认是否允许用于客户送样。

## 工艺准备

- 制作工艺卡，明确每道工序的操作方法、关键参数、检查项目和责任人。
- 裁线工序需设置总线长、分支长度和剥线长度的首件确认标准。
- 屏蔽层处理需明确是否单端接地、双端连接、浮空或连接到指定 pin，并记录实际处理方式。
- 压接工序需定义端子型号、压接高度、拉力要求、芯线外露长度和压接外观判定标准。
- 焊接工序需定义焊锡量、焊点外观、绝缘距离、热缩保护和防虚焊检查方法。
- 分线盒装配需控制位置、方向、应力释放和内部线序，防止装配后分支长度偏移。
- 标签粘贴或套管安装需规定位置、方向、内容和耐久性检查方法。

## 治具与测试设备准备

- 导通测试治具需覆盖所有设备端 pin 与患者端导联的对应关系，测试程序中的 pin 定义必须由项目工程师复核。
- 防呆治具需确认连接器插入方向，避免因针脚编号视角错误导致 pin 对应关系反向。
- 患者端接触测试夹具需能稳定接触 snap / clip / banana，不应因夹具接触不良误判样品异常。
- 尺寸测量工具需校验有效，包括卡尺、钢尺、定位板或线束展开检具。
- 绝缘电阻和耐压测试设备需确认量程、测试电压、保持时间和校准状态。
- 实机 ECG 设备或模拟器需确认型号、软件版本、导联设置、滤波设置和测试账号。
- 运动噪声观察需准备简单的摆动、弯折或轻拉动作规范，保证不同样品的观察条件一致。

## 样品制作过程检查

- 裁线后检查总线长、每个分支长度、分线盒预留位置和剥线长度是否符合图纸。
- 芯线分配前核对颜色、导联名称和设备端 pin 定义，防止 RA / LA / LL / RL / V 等导联错接。
- 设备端端子装入壳体后进行 100% pin 位复核，确认端子未退针、未错位、未反装。
- 患者端 snap / clip / banana 装配后检查压接或焊接强度，确认接触件无松动、歪斜、污染或镀层损伤。
- 屏蔽层处理完成后记录连接点、长度、包覆方式和绝缘保护，禁止只口头说明。
- 分线盒组装前后分别确认位置，避免注塑、灌封或热缩后无法再修正。
- 每根样品完成后拍摄关键工序照片，至少包含设备端、患者端、分线盒、标签和整体展开状态。

## 外观与尺寸检查

- 检查设备端连接器外观，无破损、变形、毛边、端子外露、锁扣异常或污染。
- 检查患者端 snap / clip / banana 外观，无松动、压伤、氧化、镀层脱落、尖锐边缘或插拔卡滞。
- 检查线材外被，无划伤、压痕、露屏蔽、起泡、脏污、色差或明显扭曲。
- 测量总线长、各分支长度、分线盒位置、标签位置和患者端尾部应力释放长度。
- 确认分线盒方向与图纸一致，导联分出顺序和左右方向不得与图纸相反。
- 确认 AHA / IEC 颜色、导联文字、标签内容、样品编号和警示标识一致，不允许颜色体系混用。
- 检查样品编号是否清晰、牢固、唯一，并能对应 EVT 版本、BOM、图纸和测试记录。

## 电气功能检查

- 按导通测试表逐一确认设备端 pin 到患者端 snap / clip / banana 的对应关系。
- 重点复核设备端 pin 定义，确认信号 pin、参考 pin、屏蔽 / GND pin 和空 pin 没有错接。
- 检查开路、短路、交叉线、端子未压入、屏蔽层误接到信号线等异常。
- 测量患者端接触电阻，snap / clip / banana 在正常连接状态下应稳定，不应轻微晃动就跳变。
- 测量绝缘电阻，确认相邻芯线、芯线对屏蔽、芯线对外露金属件满足 EVT 判定标准。
- 如项目要求耐压测试，记录测试电压、保持时间、漏电流和是否击穿。
- 测试完成后在每根样品记录中标明测试设备编号、测试程序版本和测试人员。

## ECG 实机连接检查

- 使用目标 ECG 设备或客户指定设备连接样品，记录设备型号、软件版本、导联模式和滤波设置。
- 确认设备端连接器插入顺畅、锁扣有效、不会因轻拉导致断开或接触不良。
- 确认患者端 snap / clip / banana 与电极片或附件连接稳定，轻微移动时不应频繁掉线。
- 查看各导联 ECG 波形是否能正常显示，RA / LA / LL / RL / V 等导联位置应与设备显示一致。
- 观察静止状态下波形是否稳定，是否存在明显基线漂移、毛刺、饱和、掉导联提示或随机噪声。
- 进行轻微摆动、弯折、翻身模拟或拉动线缆时，观察运动噪声是否明显增加并记录发生位置。
- 若实机波形异常，必须同步检查 pin 定义、患者端接触、屏蔽连接、设备设置和电极片状态。

## 屏蔽与抗干扰初步确认

- 核对屏蔽层实际连接方式与图纸或设计假设一致，例如单端接设备 GND、双端连接或浮空。
- 检查屏蔽层在设备端、分线盒和患者端是否存在毛丝外露、短接信号线或绝缘保护不足。
- 对比不同屏蔽连接方案样品的实机波形，记录静止噪声、运动噪声和外部干扰下的差异。
- 在靠近电源适配器、显示器、手机或其他常见干扰源时观察波形变化，作为初步抗干扰判断。
- 轻微弯折设备端尾部、分线盒出口和患者端尾部，观察是否因屏蔽断续接触产生噪声尖峰。
- 将屏蔽验证结果写入 EVT 总结，说明当前方案是否可进入下一阶段，或需要调整接地点和包覆方式。

## 机械可靠性初步确认

- 对设备端连接器进行插拔手感确认，检查锁扣、端子保持力和插入方向防呆是否正常。
- 对患者端 snap / clip / banana 进行多次插拔，确认接触稳定、夹持力或扣合力无明显衰减。
- 对线缆进行轻拉测试，重点观察设备端尾部、分线盒出口、患者端尾部和标签区域。
- 对分支线进行低次数弯折或摆动，观察外被开裂、导通中断、波形毛刺和接触噪声。
- 检查分线盒、热缩管、护套和应力释放结构是否移位、开胶、翘边或影响线缆柔软度。
- 机械测试后的样品不得直接送客户，除非记录中明确标注测试条件和样品状态。

## 消毒与清洁初步确认

- 按客户或项目预期清洁方式进行初步擦拭，例如酒精、季铵盐类清洁剂或指定医用清洁剂。
- 检查线材外被、分线盒、热缩管、标签和印字是否出现褪色、发粘、开裂、起泡或脱落。
- 检查患者端 snap / clip / banana 接触面清洁后是否残留液体、腐蚀痕迹或接触不稳定。
- 清洁后再次进行导通和患者端接触稳定性检查，确认没有因清洁导致瞬断或电阻异常。
- 记录清洁剂名称、浓度、擦拭次数、干燥时间和检查结果，避免只写“消毒 OK”。
- 对标签耐久性重点确认，样品编号和导联标识清洁后仍需清晰可读。

## 客户送样资料

- 送样清单需列明样品编号、数量、版本、图纸版本、BOM 版本、制作日期和测试状态。
- 提供 EVT 图纸或客户允许查看的关键尺寸资料，确保客户能核对线长、分支长度、分线盒位置和接口形式。
- 提供 pin 定义确认表，明确设备端每个 pin 对应的导联、屏蔽 / GND 或空位。
- 提供 AHA 或 IEC 颜色定义说明，注明本批样品采用哪一套颜色体系。
- 提供样品全检记录，包括外观、尺寸、导通、绝缘、患者端接触和实机 ECG 波形观察结果。
- 提供异常清单和未关闭事项，说明哪些问题已修正，哪些问题仍需客户共同评估。
- 包装标签需包含项目名称、样品版本、样品编号、数量、制作日期和联系人。
- 若样品仅用于 EVT 评估，需在送样资料中注明“工程样品，非量产状态”。

## EVT 总结与下一步

- 汇总 EVT 样品制作结果，包括计划数量、实际完成数量、客户送样数量、留样数量和报废数量。
- 汇总关键验证结论，重点说明设备端 pin 定义、患者端接触稳定性、AHA / IEC 颜色体系、线长尺寸和屏蔽连接是否通过初步确认。
- 汇总实机 ECG 连接结果，说明静止波形、运动噪声、掉导联提示和异常波形是否满足进入下一阶段的条件。
