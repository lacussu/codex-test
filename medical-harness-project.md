# 医疗线束项目工程师学习项目

## 项目背景

1. 这个医疗线束项目用于哪类设备或系统？
2. 客户或项目方对线束的主要功能要求是什么？
3. 项目的应用环境有哪些特殊要求，例如温度、湿度、消毒或移动使用？
4. 当前项目处于哪个阶段，例如样品、验证、小批量或量产？

## CAD 图纸内容

1. 图纸中线束总长、分支长度和公差分别是多少？
2. 图纸中连接器、端子、护套、标签和套管的位置是否明确？
3. 图纸中是否标注了线号、颜色、线径和走线路径？
4. 哪些尺寸、视图或标注需要进一步确认？

## BOM 清单

1. BOM 中包含哪些主要材料，例如线材、连接器、端子、热缩管和标签？
2. 每种材料的规格、型号、品牌或替代料是否清楚？
3. 关键材料是否有医疗行业相关要求或客户指定要求？
4. BOM 数量、单位和损耗是否需要进一步核对？

## Pin 位表

1. 每个连接器的 Pin 位编号和定义是否完整？
2. 线材两端的连接关系是否一一对应？
3. 是否存在空 Pin、短接、屏蔽线或特殊接地要求？
4. Pin 位表与 CAD 图纸、BOM 是否一致？

## 工艺要求

1. 端子压接、焊接、插壳、包扎和热缩的工艺要求分别是什么？
2. 压接高度、拉力、剥线长度等关键参数是否明确？
3. 是否需要防呆、极性识别、颜色区分或标签追溯？
4. 生产过程中哪些步骤最容易出错，需要重点控制？

## 检验要求

1. 需要进行哪些检验项目，例如外观、导通、耐压、绝缘和拉力测试？
2. 每个检验项目的判定标准和记录方式是什么？
3. 是否需要首件检验、过程检验或出货检验？
4. 不合格品的隔离、返工和复检流程是什么？

## 风险清单

1. 这个项目在设计、材料、工艺和检验方面有哪些主要风险？
2. 哪些风险会影响医疗设备的安全性、可靠性或合规性？
3. 每个风险的发生原因、影响程度和预防措施是什么？
4. 哪些风险需要提前与客户、供应商或生产团队确认？

## 后续学习计划

1. 我接下来需要重点学习哪些医疗线束基础知识？
2. 哪些 CAD、BOM、Pin 位表或工艺文件还需要继续练习？
3. 我需要向哪些同事、供应商或资料来源请教？
4. 下次复盘时，我希望自己能解决哪些具体问题？
