# ECG 心电线风险、测试与项目模板总表

## 文件目的

- 给医疗线束项目工程师新人提供一份可直接复制到项目文件夹的 ECG 心电线项目总模板。
- 把风险管理、测试计划、BOM、Pin 位、样品、客户沟通和未关闭问题统一到一个文件中，避免信息分散。
- 帮助项目工程师在立项、打样、EVT、DVT、PVT、量产导入和变更阶段持续追踪关键风险。
- 确保 ECG 心电线的导联颜色、Pin 定义、患者端接触、电气性能、机械寿命、清洁消毒、包装和标签追溯均有记录。
- 作为项目会议、客户评审、内部设计评审、样品评审和测试评审的基础检查清单。

## 使用场景

- 项目启动时，用于建立 ECG 心电线项目主控表，确认客户规格、适用标准、导联数量和接口形式。
- 设计输入评审时，用于核对 AHA / IEC 颜色体系、设备端 Pin 定义、患者端 snap / clip / banana 接口要求。
- 样品制作前，用于确认 BOM、图纸、Pin 位表、线色、标签内容、包装方式和关键物料来源。
- EVT / DVT 阶段，用于安排接触电阻、屏蔽连续性、运动噪声、弯折寿命、插拔寿命和酒精擦拭测试。
- 客户送样时，用于记录样品编号、版本、数量、用途、测试状态和客户反馈。
- 项目量产导入前，用于关闭未解决问题，确认测试结论、客户批准、检验标准和追溯方式。
- 工程变更时，用于评估材料、接口、线色、Pin 位、包装、标签或工艺变更带来的风险。

## ECG 项目主要风险

- AHA / IEC 颜色体系混用：确认客户要求使用 AHA 还是 IEC，检查导联名称、线色、标签和说明文件是否一致。
- 设备端 Pin 定义错误：确认客户设备端连接器型号、Pin 序、视图方向、焊接端方向和图纸标注方向。
- 患者端接口接触不稳定：确认 snap、clip、banana 与电极片或适配器的配合尺寸、夹持力、插拔手感和接触电阻。
- 屏蔽连续性不足：确认屏蔽层覆盖率、接地端连接方式、屏蔽断点、分叉处屏蔽处理和端子压接可靠性。
- 运动噪声过大：确认线材柔软度、导体结构、屏蔽结构、分叉长度、应力释放和患者端接口晃动噪声。
- ECG 实机波形不稳定：确认在目标设备上是否出现基线漂移、毛刺、断线提示、导联识别异常或波形饱和。
- 接触电阻偏高：确认设备端端子、患者端接口、压接点、焊点和导体之间的总电阻是否满足客户要求。
- 弯折寿命不足：确认主线、分叉线、SR 应力释放、连接器尾部和患者端尾部是否能通过弯折测试。
- 插拔寿命不足：确认设备端连接器和患者端 snap / clip / banana 在规定次数后无松动、破损、镀层异常或接触不良。
- 耐酒精擦拭不足：确认线材外被、标签、印字、颜色环、胶壳和患者端接口经酒精擦拭后无脱色、开裂、发粘或字迹模糊。
- 标签追溯不完整：确认型号、批号、生产日期、版本、UDI 或客户要求信息是否能追溯到物料批次和生产记录。
- 包装导致线材弯折变形：确认绕线直径、扎带位置、吸塑或袋装压线风险，避免长期包装后线材折痕影响外观和噪声。
- 客户规格不完整：确认客户是否明确导联体系、线长、颜色、接口、材料、测试标准、包装、标签和法规要求。
- 关键物料替代风险：确认线材、连接器、端子、电极扣、夹子、标签、油墨和包装材料变更是否需要客户批准。
- 工艺一致性风险：确认剥线、压接、焊接、注塑、包胶、屏蔽处理、导通测试和外观检验是否有作业标准。

## 电气测试项目

- 导通测试：逐 Pin 检查设备端到患者端对应关系，确认 RA、LA、LL、RL、V 或 C 系列导联无错位。
- 短路测试：检查相邻 Pin、导联之间、导体与屏蔽之间是否存在短路。
- 接触电阻测试：分别测试设备端连接器、患者端 snap / clip / banana 和整线回路电阻。
- 接触电阻稳定性：插拔后、弯折后、晃动后重复测量，记录最大值、最小值和波动范围。
- 绝缘电阻测试：按客户规格确认导体间、导体对屏蔽、导体对外表面的绝缘能力。
- 耐压测试：按适用标准或客户要求确认无击穿、闪络、漏电超限。
- 屏蔽连续性测试：检查屏蔽层是否按设计连接到设备端指定 Pin、壳体或地线。
- 屏蔽隔离检查：确认不应连接的位置无误接，避免屏蔽与信号导体短接。
- 端子压接电气确认：抽样检查压接截面、电阻和拉力，避免虚压、断丝或压接过深。

## 机械可靠性测试项目

- 弯折寿命测试：重点覆盖设备端尾部、主线分叉处、患者端尾部和最小弯曲半径区域。
- 插拔寿命测试：分别对设备端连接器和患者端 snap / clip / banana 执行规定次数插拔。
- 拉力测试：确认主线、分叉线、端子、患者端扣件、夹子和包胶区域不脱落。
- 扭转测试：确认线材在使用中旋转后不出现导体断裂、护套开裂或屏蔽断裂。
- 摆动测试：模拟患者活动时导联线反复摆动，检查导通、噪声和外观。
- 分叉强度检查：确认分叉点注塑、热缩管或扎线结构不裂开、不滑移。
- 外观磨损检查：对线身、印字、颜色环、连接器外壳、患者端接口做摩擦后检查。
- 包装压线检查：模拟库存和运输后确认线材无严重压痕、折白、永久变形或护套破损。

## 消毒与清洁测试项目

- 酒精擦拭测试：按客户规定浓度和次数擦拭线身、标签、连接器外壳和患者端接口。
- 标签耐擦拭测试：确认型号、批号、方向标识、导联标识和追溯码擦拭后仍清晰。
- 印字耐擦拭测试：确认线身印字、颜色标识和客户 Logo 不脱落、不扩散。
- 材料外观检查：擦拭后检查 PVC、TPU、TPE、硅胶或胶壳是否发粘、发白、开裂、变硬。
- 接触性能复测：清洁测试后复测接触电阻、导通和患者端接口配合稳定性。
- 清洁剂兼容性确认：如客户使用异丙醇、乙醇、季铵盐或其他消毒剂，应分别确认材料兼容性。

## ECG 实机功能测试项目

- 设备识别检查：连接目标 ECG 设备，确认设备无导联脱落、连接异常或附件不兼容提示。
- 波形稳定性检查：观察静止状态下波形基线是否平稳，有无毛刺、跳变、漂移或间歇断线。
- 运动噪声检查：轻微摆动主线、分叉线和患者端接口，确认波形噪声在客户可接受范围内。
- 导联对应检查：逐一触碰或模拟输入各导联，确认设备显示通道与 Pin 位表一致。
- 患者端接口检查：分别验证 snap、clip、banana 与客户指定电极片或适配器的接触稳定性。
- 屏蔽效果观察：在常见干扰环境下比较屏蔽连接前后噪声表现，记录测试环境和设备设置。
- 长时间运行检查：连续连接运行，确认无温升异常、接触间歇、波形漂移或设备报警。
- 送样前确认：送客户样品必须完成导通、Pin 位、外观、标签、包装和实机波形确认。

## 未关闭问题清单

- AHA / IEC 颜色体系是否已有客户书面确认。
- 设备端连接器型号、Pin 数、键位和视图方向是否已经冻结。
- Pin 位表是否由工程、品质、生产和客户四方确认。
- 患者端 snap / clip / banana 是否已与客户指定电极片完成配合测试。
- 屏蔽层连接方式是否已经在图纸和作业指导书中明确。
- 运动噪声测试方法、动作幅度、判定标准和记录方式是否已确定。
- ECG 实机测试是否覆盖目标设备型号、软件版本和典型使用场景。
- 接触电阻限值是否明确，测试点位和测试夹具是否统一。
- 弯折寿命和插拔寿命次数是否来自客户规格或内部标准。
- 酒精擦拭测试是否覆盖线身、标签、印字、连接器和患者端接口。
- 标签追溯字段是否能关联到生产批次、物料批次、检验记录和出货记录。
- 包装绕线直径是否足够，是否存在扎带压伤、折痕、吸塑压线或运输变形风险。
- 关键物料清单是否已经冻结，替代料是否需要客户批准。
- 量产检验项目是否包括导通、短路、Pin 位、外观、标签和包装检查。
- 客户反馈是否全部转化为责任人、截止时间和关闭证据。

## 项目工程师使用建议

- 项目开始第一天就建立本文件，不要等到 DVT 失败后再补风险表。
- 每次客户会议后，立即更新客户沟通记录表，并把待关闭事项转入未关闭问题清单。
- 所有 ECG 项目必须优先确认三件事：颜色体系、设备端 Pin 定义、患者端接口类型。
- Pin 位表不能只看客户文字描述，必须确认连接器正视图、焊接视图、线束端视图是否一致。
- 样品送客户前，至少完成导通、短路、Pin 位、外观、标签、包装和 ECG 实机波形确认。
- 对运动噪声和波形稳定性，不要只依赖万用表测试，必须在目标设备或等效 ECG 设备上验证。
- 对屏蔽连续性，既要测电阻，也要检查分叉处和端部实际工艺，必要时拆解首件确认。
- 对酒精擦拭，必须把标签和印字作为重点，因为追溯信息模糊会直接影响质量审核。
- 对包装，要用实际出货方式放置一段时间后再检查线材折痕，不只看刚包装完成的状态。
- 风险登记表中的高风险项目必须有关闭证据，例如客户邮件、测试报告、照片、图纸版本或检验记录。
- BOM 中标为关键物料的项目，任何型号、供应商、材料、镀层、颜色或工艺变化都应启动变更评估。
- 量产前组织一次跨部门关闭会议，逐项确认风险、测试、样品、客户反馈和未关闭问题是否全部完成。
