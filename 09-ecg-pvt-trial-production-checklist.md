# ECG 心电线小批试产 PVT 清单

## PVT 的目的

- 确认 ECG 心电线从样品验证状态进入小批生产时，生产线、工艺、人员、物料、治具、测试方法和质量控制是否具备稳定复制能力。
- 验证设备端连接器 pin 定义、患者端 snap / clip / banana 接触方式、线材分支结构、屏蔽层处理和标签追溯在批量条件下是否一致。
- 通过小批试产数据确认关键尺寸、电气性能、ECG 实机波形、运动噪声、接触电阻和包装状态是否满足客户图纸、BOM、规格书和验证要求。
- 暴露量产前的工艺波动、物料匹配、作业培训、检验标准、返工流程和异常闭环问题，为量产导入提供依据。

## PVT 前置条件

- 客户图纸、产品规格书、BOM、线序表、pin 定义表、AHA / IEC 颜色体系要求已受控发行，且版本一致。
- EVT、DVT 阶段遗留问题已关闭，或已形成经客户 / 项目 / 质量批准的临时控制措施。
- 设备端连接器、患者端端子、线材、屏蔽材料、外被、标签、包装材料均已完成供应商确认和来料规格确认。
- 生产流程、关键工序参数、检验标准、测试方法、抽样方案和不良判定标准已评审通过。
- ECG 实机、模拟器、电气测试仪、接触电阻测试治具、拉力测试设备、尺寸量具等已完成校准或点检。
- 试产使用的人员、设备、工装、物料和场地应尽量接近量产状态，不能使用未经批准的临时替代方法。
- 项目工程师已建立 PVT 试产问题清单、良率统计表、异常记录表和样品追溯表。

## PVT 输入资料

- 客户最新受控图纸，包括总长、分支长度、线径、连接器方向、端子类型、标签位置和包装要求。
- 产品规格书，包括电气性能、接触电阻、绝缘电阻、耐压、屏蔽要求、运动噪声、机械可靠性和外观要求。
- 设备端 pin 定义表，需明确每个 pin 对应 RA、LA、LL、RL、V / C 等导联信号，防止线序接错。
- AHA / IEC 颜色定义表，需明确本项目采用哪一种颜色体系，不允许同一批产品混用。
- BOM 和 AVL，需明确线材、导体、屏蔽层、连接器、snap / clip / banana、标签、胶壳、热缩管和包装材料料号。
- 工艺流程图、PFMEA、控制计划、SOP、检验指导书、测试作业指导书和包装规范。
- EVT / DVT 问题关闭清单、设计变更记录、客户确认邮件或会议纪要。
- PVT 样品用途分配表，包括内部测试、客户送样、可靠性抽检、留样、破坏性测试和首批量产参考样。

## 试产数量与用途

- 试产数量应覆盖生产节拍、换线、人员操作、关键尺寸波动和测试抽检需要，不能只做少量展示样。
- 建议将试产样品按用途分配：客户送样、内部全检样、ECG 实机抽检样、机械可靠性抽检样、包装运输评估样、留样和不良分析样。
- 每条样品需记录唯一编号、生产日期、物料批号、作业员、检验员、测试结果和最终去向。
- 破坏性测试样品需单独标识，不能混入客户送样或可交付样品。
- 若客户指定 PVT 数量、抽样比例或交付批次，项目工程师需在试产计划中单独列出并跟进。

## 生产线准备

- 确认生产线按 ECG 心电线工艺流程布置，包含裁线、剥线、屏蔽处理、焊接 / 压接、注塑 / 组装、导通测试、外观检验、包装等工位。
- 确认各工位有受控 SOP、图纸、线序图、颜色对照表和不良样板，现场版本需与项目受控版本一致。
- 确认设备端连接器组装方向、防呆方式和 pin 位识别清楚，避免 pin 1 方向或镜像方向错误。
- 确认患者端 snap / clip / banana 工位有接触面保护措施，避免镀层划伤、变形、松动或污染。
- 确认线材分支定位治具可稳定控制总长、分支长度和分叉角度。
- 确认屏蔽层修剪、翻折、接地、包覆、热缩或铜箔处理工位有明确工艺边界，避免每人做法不同。
- 确认半成品周转方式不会造成线材打结、过度弯折、端子污染或标签脱落。

## 人员培训

- 培训作业员识别 ECG 导联名称、AHA / IEC 颜色差异、设备端 pin 定义和患者端端子类型。
- 培训作业员按 SOP 控制剥线长度、屏蔽层长度、焊接温度、压接高度、热缩位置、标签方向和包装绕线方式。
- 培训检验员按图纸测量总长、分支长度、连接器方向、标签位置和端子外观。
- 培训测试员正确连接导通测试治具、接触电阻夹具、ECG 模拟器和实机测试设备。
- 培训 IPQC 识别关键不良：线序错、pin 位错、颜色体系混用、端子接触不稳、屏蔽短路、屏蔽断开、分支长度超差、外被损伤。
- 培训所有相关人员填写批号、工序记录、异常单、返工记录和样品追溯表。
- 培训完成后需保留签到表、考核记录和首批上岗人员名单。

## 物料准备

- 核对线材规格、芯线颜色、屏蔽结构、外被材质、线径、柔软度和供应商批号。
- 核对设备端连接器型号、pin 数、键位方向、外壳颜色、插拔配合和端子镀层。
- 核对患者端 snap / clip / banana 端子型号、颜色、接触面状态、弹性和插拔手感。
- 核对 AHA 或 IEC 颜色体系对应的导联颜色，确认同一订单、同一批次、同一包装内不混用。
- 核对屏蔽材料、铜箔、编织层、排流线、热缩管、套管、胶水和固定材料是否与 BOM 一致。
- 核对标签内容、标签材质、打印方式、耐擦拭要求、批号规则和贴附位置。
- 核对包装袋、绕线方式、内托、隔离纸、外箱和运输标签，确保包装不会造成线材弯折变形。
- IQC 需对关键物料做来料检验并记录批号，未经放行物料不得用于 PVT。

## 工艺文件准备

- 确认工艺流程图覆盖从裁线到包装的全部工序，且与现场实际路线一致。
- 确认 SOP 中有明确的剥线尺寸、屏蔽层处理长度、焊接 / 压接参数、热缩位置、分支定位方式和包装绕线直径。
- 确认线序图清楚标注设备端 pin 位、导联名称、芯线颜色、患者端端子颜色和端子类型。
- 确认检验指导书包含外观缺陷等级、尺寸公差、端子接触要求、标签要求和包装判定标准。
- 确认测试指导书包含导通、短路、绝缘、耐压、接触电阻、运动噪声和 ECG 实机波形确认方法。
- 确认控制计划列出关键工序、关键参数、检验频率、责任人和记录表单。
- 确认所有文件有版本号、发行日期、批准人和受控标识，现场不能使用草稿或旧版本。

## 治具与测试设备准备

- 确认裁线、剥线、分支定位、焊接、压接、热缩、端子组装和包装治具状态良好。
- 确认设备端 pin 位测试治具能覆盖全部导联，不允许只测导通而忽略 pin 定义对应关系。
- 确认患者端 snap / clip / banana 接触稳定性测试夹具与产品端子匹配，不会因夹具松动误判。
- 确认导通 / 短路测试仪、绝缘电阻测试仪、耐压仪、接触电阻仪、拉力机和卡尺在校准有效期内。
- 确认 ECG 实机或 ECG 模拟器可正常工作，测试导联、参考电极、测试环境和判定样张已准备。
- 确认运动噪声测试方法可执行，例如轻微摆动、弯折、扭转或模拟患者移动时观察波形噪声。
- 确认测试设备每日点检项目、点检频率、点检人和异常处理方式已明确。

## 首件确认

- 首件需由生产按正式工艺制作，不能由工程师绕过正常工序手工特制。
- 首件需确认设备端连接器方向、pin 1 位置、每个 pin 对应导联和患者端颜色 / 端子是否完全一致。
- 首件需确认 AHA 或 IEC 颜色体系无混用，标签、包装和检验记录也需使用同一体系名称。
- 首件需测量总长、各分支长度、分叉点位置、标签位置、热缩位置和关键外形尺寸。
- 首件需确认屏蔽层处理方式，包括保留长度、排流线连接、绝缘隔离、热缩覆盖和短路风险。
- 首件需完成导通、短路、绝缘、耐压、接触电阻和 ECG 实机波形确认。
- 首件确认通过后，项目、工程、质量和生产需签字放行；未通过不得继续批量生产。

## 外观与尺寸检验

- 检查线材外被无破皮、压痕、脏污、缩水、鼓包、露芯、明显色差和过度扭曲。
- 检查设备端连接器无变形、毛边、缩水、裂纹、pin 缩针、pin 歪斜、异物和装配间隙异常。
- 检查患者端 snap / clip / banana 无镀层划伤、弹片变形、松动、污染、毛刺和接触面氧化。
- 检查导联颜色与项目定义一致，AHA / IEC 颜色体系不可混用或标签错标。
- 测量总长、每个分支长度、分叉点位置、标签位置、热缩位置和端子外露尺寸。
- 检查分支长度稳定性，确认同批产品不会出现明显长短不一、左右导联交叉或自然垂落差异过大。
- 检查所有尺寸数据是否记录到尺寸检验表，超差样品需隔离并分析原因。

## 电气测试

- 100% 确认设备端每个 pin 与对应患者端导联导通，特别关注 RA、LA、LL、RL、V / C 等导联对应关系。
- 100% 确认任意不同导联之间无短路，屏蔽层与信号芯线之间无异常短接。
- 按规格要求测试绝缘电阻，确认线材、连接器、端子和屏蔽隔离满足要求。
- 按规格要求测试耐压，确认无击穿、闪络或测试后性能异常。
- 抽检患者端 snap / clip / banana 接触电阻，关注插拔后、轻微晃动后和不同接触角度下的稳定性。
- 抽检运动噪声，轻微摆动或弯折线材时观察电气信号是否出现异常跳变。
- 所有测试不合格样品需贴不良标签、隔离存放，并记录测试项目、数值和设备编号。

## ECG 实机功能确认

- 使用客户指定或项目认可的 ECG 实机 / ECG 模拟器进行抽检确认。
- 确认设备端插入顺畅、锁扣可靠、pin 位识别正确，实机能正确识别导联。
- 确认患者端 snap / clip / banana 与电极片、夹具或模拟端连接稳定，无松脱和接触不良。
- 确认各导联波形显示正常，无导联反接、导联缺失、基线漂移异常或明显噪声。
- 轻微移动线材、摆动患者端端子和弯折分支处，观察波形是否出现异常尖峰、断续或接触噪声。
- 若客户要求 AHA / IEC 导联标识显示，需确认设备显示、线材颜色和标签标识一致。
- 保存 ECG 波形截图、测试设备型号、测试日期、样品编号和测试人员记录。

## 屏蔽与抗干扰确认

- 确认屏蔽层处理方式与图纸和 SOP 一致，包括单端接地 / 双端处理、排流线连接点和绝缘隔离方式。
- 检查屏蔽层修剪后无散丝刺穿绝缘层、无屏蔽丝接触信号芯线、无排流线虚焊或断开。
- 抽检屏蔽连续性，确认屏蔽回路未因裁切、焊接、热缩或注塑过程断开。
- 抽检屏蔽与信号芯线之间的绝缘状态，避免屏蔽短路导致波形异常。
- 在 ECG 实机或模拟测试环境中观察抗干扰表现，关注工频干扰、运动噪声和基线漂移。
- 对屏蔽处理不一致的样品进行分层统计，确认是否与作业员、班次、物料批号或治具状态相关。

## 机械可靠性抽检

- 抽检设备端连接器插拔力、锁扣保持力和插拔后 pin 位稳定性。
- 抽检患者端 snap / clip / banana 插拔力、夹持力、弹性恢复和接触稳定性。
- 抽检线材弯折，重点关注设备端尾部、分叉点、患者端尾部和热缩边缘。
- 抽检端子拉力或焊接 / 压接保持力，确认无松脱、断线、虚焊和压接不良。
- 抽检分支处抗拉和抗扭表现，确认分支长度和外观不会因轻微拉扯明显变化。
- 机械抽检后的样品需复测导通、短路、接触电阻和 ECG 波形，确认可靠性测试后功能仍正常。
- 破坏性测试样品需单独编号并隔离，不得作为客户可交付样品。

## 标签与追溯确认

- 确认标签内容包含客户要求的信息，例如料号、批号、生产日期、导联体系、方向标识、UDI / SN 或内部追溯码。
- 确认标签批号与生产记录、物料批号、测试记录、包装记录和样品编号可相互追溯。
- 检查标签位置、方向、粘贴牢固度、可读性、条码扫描和耐擦拭性能。
- 确认 AHA / IEC 标识与实际线材颜色体系一致，不能因标签模板错误造成混用风险。
- 确认返工品重新测试后仍保留原始批号和返工记录，不能丢失追溯链。
- 留样、客户送样、测试样和报废样都需在样品追溯表中记录最终去向。

## 包装确认

- 确认包装方式符合客户规范，包括绕线直径、扎带位置、袋型、标签位置、说明书、内盒和外箱要求。
- 确认绕线半径足够大，不会造成线材永久弯折、分支折痕、外被压伤或端子受力变形。
- 确认患者端 snap / clip / banana 接触面有保护，避免运输中互相刮伤或挤压变形。
- 确认设备端连接器有防尘、防刮或防 pin 位受压措施。
- 确认同一包装内颜色体系一致，AHA 与 IEC 产品不能混装。
- 确认包装标签批号、数量、料号和样品状态与内装实物一致。
- 必要时进行模拟运输、跌落或堆码评估，开箱后检查线材是否弯折变形、标签脱落或端子损伤。

- 良率需按工序统计，不只统计最终成品良率。

- PVT 良率未达到量产目标时，需明确是否继续追加 PVT、限制量产放量或增加临时检验。

## 不良品分析

- 将不良品按电气、外观、尺寸、功能、包装、标签和物料类别分类隔离。
- 对设备端 pin 错、导联反接、颜色体系混用等重大不良，需立即停线并追溯已生产数量。
- 对患者端接触不稳和接触电阻偏高，需检查端子镀层、弹性、压接 / 焊接质量、污染和测试夹具匹配性。
- 对分支长度和总长不稳定，需检查裁线设备、定位治具、作业拉伸、热缩收缩和测量方法。
- 对屏蔽相关不良，需拆解确认屏蔽丝、排流线、焊点、绝缘隔离和热缩覆盖状态。
- 对 ECG 波形噪声，需区分产品问题、测试环境问题、实机连接问题和患者端接触问题。
- 不良分析需保留照片、测试数据、拆解记录、原因判断和纠正预防措施。

## 返工与让步处理

- 返工前需由工程和质量确认返工方法、风险、适用范围和复检项目。
- 设备端 pin 定义错误、导联反接、颜色体系混用等涉及安全和功能的不良，原则上不得直接让步放行。
- 返工后的产品需重新执行相关电气测试、外观检验、尺寸确认和必要的 ECG 实机波形抽检。
- 返工品需有返工标识和返工记录，记录原始不良、返工人员、返工日期、复检结果和最终判定。
- 对外观轻微不良或包装偏差申请让步时，需确认不影响功能、安全、追溯和客户使用体验。
- 所有让步需经项目、质量、工程和客户要求的批准层级确认后才可交付。

## PVT 结论

- 汇总 PVT 投入数量、合格数量、不良数量、一次良率、返工后良率和最终可交付数量。
- 确认设备端 pin 定义、患者端接触稳定性、AHA / IEC 颜色体系、分支长度、屏蔽处理和 ECG 实机波形是否全部满足要求。
- 确认所有客户送样、内部留样、可靠性抽检样和破坏性测试样的编号、状态和去向。
- 确认所有重大异常已关闭；未关闭项需有风险评估、临时控制措施、责任人和完成日期。
- 确认 PVT 结果是否允许进入量产：通过、条件通过、需追加 PVT 或不通过。
- PVT 结论需由项目、工程、质量、生产、采购 / 供应链和必要的客户代表共同评审确认。

## 量产导入准备

- 将 PVT 中确认有效的工艺参数、检验频率、测试方法和包装方式更新到正式量产文件。
- 将 PVT 异常和改善措施更新到 PFMEA、控制计划、SOP、检验指导书和培训资料。
- 确认量产物料供应能力、批次追溯规则、关键物料安全库存和替代料审批流程。
- 确认量产线治具、测试设备、备件、校准计划和日常点检表已准备齐全。
- 确认量产首批仍需加强控制的项目，例如 pin 定义 100% 核对、患者端接触电阻抽检、屏蔽连续性抽检和 ECG 波形抽检。
- 确认包装方式已通过试产验证，量产包装不会造成线材弯折变形、端子刮伤或标签脱落。
- 确认项目工程师建立量产爬坡跟踪计划，跟进首三批良率、客户反馈、返工率和异常关闭状态。
- 量产导入前召开 PVT 评审会议，形成会议纪要和量产放行决议。
